# Dashboard de Vendas em Excel

## Descrição do Projeto
Este projeto consiste em um **dashboard de vendas desenvolvido no Excel**, com foco na organização e visualização de dados. O objetivo é transformar dados brutos em informações visuais claras e úteis, permitindo a análise do desempenho de vendas e a tomada de decisões baseadas em dados.

O dashboard apresenta KPIs, gráficos interativos e filtros que facilitam a interpretação das informações e permitem insights rápidos sobre vendas por produto, região, vendedor e período.

---

## Base de Dados
O projeto utiliza o arquivo `base.xlsx`, que contém os dados de vendas com as seguintes colunas:

- **Data Venda:** Data da venda
- **Produto:** Nome do produto vendido
- **Categoria:** Categoria do produto
- **Quantidade:** Quantidade vendida
- **Valor Venda:** Valor monetário da venda
- **Região:** Região onde a venda foi realizada
- **Vendedor:** Nome do vendedor responsável pela venda

> Certifique-se de manter os tipos de dados corretos (datas como Data, valores como Número) para o funcionamento correto do dashboard.

---

## Estrutura do Dashboard
O dashboard (`dashboard_vendas.xlsx`) possui os seguintes elementos:

### KPIs Principais
- **Total de Vendas** – soma do valor de todas as vendas
- **Total de Produtos Vendidos** – soma das quantidades vendidas
- **Ticket Médio** – valor médio por venda
- **Número de Vendedores** – total de vendedores registrados

### Gráficos
- **Linha do Tempo de Vendas:** vendas por mês
- **Barras de Vendas por Produto:** comparação entre produtos
- **Pizza de Vendas por Região:** participação de cada região
- **Tabela Resumida:** ticket médio por vendedor

### Filtros Interativos (Slicers)
- Período (mês/ano)
- Produto ou categoria
- Região
- Vendedor

---

## Como Usar
1. Abra o arquivo `dashboard_vendas.xlsx` no Excel.  
2. Utilize os **filtros (slicers)** para ajustar o período, produto, região ou vendedor.  
3. Observe os **KPIs** no topo para uma visão rápida do desempenho.  
4. Analise os **gráficos** para identificar tendências, padrões e áreas de destaque nas vendas.  

> Dica: todos os gráficos e KPIs são dinâmicos e se atualizam automaticamente quando os filtros são aplicados.
